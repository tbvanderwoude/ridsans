# Required specifications
To perform an accurate reduction, the following information is still needed
- Exact detector dimensions, resolution etc.
- Beamstop dimensions (small and large) as well as meaning of x, y position values (are they the centre, a cornerpoint and if so which?)
- Order of CDAT2 pixel array: does (0,0) correspond to the lowerleft, topleft, ... ?
  - Related: are the values row or column major ((0,0), (0,1) etc. or (0,0), (1,0))
